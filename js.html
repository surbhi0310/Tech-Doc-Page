DOM Manipulation
(One of the fundamental usecase of JS)

When we say maipulating the dom, we mean that using JS we should be able to control what elements are being added/removed to the DOM, how they behave in terms of user interaction, how their look and feel is and everything else related to these HTML elements.

How to acheive DOM Manipulation?

So, our browsers provide us a lot of functions and objects using which we can control whole DOM. Important thing t note is that these functions are not nativeto JS, instead they are provided by the browser (which is the runtime environment).

  2. document.getElementByClass
  3. document.getElementByTagName
  4. document.querySelector
  5. document.getElementsByClassName
  6. and more...

    7. Browsers provide us a very imp object which is globally present everywherein the browser's JS, called document.
    It Contains a lot of methods and properties using which we can manipulate the DOM.
